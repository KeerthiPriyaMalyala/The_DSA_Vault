🌳 Detect a Cycle in an Undirected Graph (DFS)
❓ What is the question asking?

We are given an undirected graph.

We need to check:

✅ Does the graph contain a cycle?

🧠 What is a Cycle?

A cycle means:

👉 We can start from a node and come back to it again through some path.

🌟 Example
Graph With Cycle
0 ----- 1
|       |
|       |
3 ----- 2

Cycle:

0 → 1 → 2 → 3 → 0

✅ Answer = TRUE

Graph Without Cycle
0 --- 1 --- 2

No cycle ❌

🧠 Main Idea

During DFS:

Visit neighbors
If neighbor is:
not visited → continue DFS
already visited AND not parent → cycle found
🌟 Why Parent Needed?

In undirected graph:

0 ↔ 1

When DFS goes:

0 → 1

Again seeing 0 from 1 is NORMAL.

That is not cycle.

So we ignore parent.

⚡ Optimal Approach (DFS)
💡 Idea
Start DFS
Pass:
current node
parent node
Traverse neighbors
If visited neighbor is not parent:
cycle exists
💻 Java Code
import java.util.*;

public class Main {

    static boolean dfs(int node,
                       int parent,
                       ArrayList<ArrayList<Integer>> graph,
                       boolean[] visited) {

        visited[node] = true;

        for(int neighbor : graph.get(node)) {

            if(!visited[neighbor]) {

                if(dfs(neighbor, node, graph, visited)) {
                    return true;
                }
            }

            else if(neighbor != parent) {

                return true;
            }
        }

        return false;
    }

    public static void main(String[] args) {

        int V = 4;

        ArrayList<ArrayList<Integer>> graph = new ArrayList<>();

        for(int i=0;i<V;i++) {
            graph.add(new ArrayList<>());
        }

        graph.get(0).add(1);
        graph.get(1).add(0);

        graph.get(1).add(2);
        graph.get(2).add(1);

        graph.get(2).add(3);
        graph.get(3).add(2);

        graph.get(3).add(0);
        graph.get(0).add(3);

        boolean[] visited = new boolean[V];

        boolean cycle = false;

        for(int i=0;i<V;i++) {

            if(!visited[i]) {

                if(dfs(i, -1, graph, visited)) {

                    cycle = true;
                    break;
                }
            }
        }

        System.out.println(cycle);
    }
}

🔥 Important Points

✅ Works for:

Undirected Graph

✅ Parent is VERY important.

✅ Cycle condition:

visited neighbor
AND
neighbor != parent
🌟 BFS vs DFS
BFS	DFS
Uses Queue	Uses Recursion/Stack
Store (node,parent) in queue	Pass parent in recursion
Same logic	Same logic
