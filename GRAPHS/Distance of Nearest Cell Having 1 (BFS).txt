🌳 Distance of Nearest Cell Having 1 (BFS)
❓ What is the question asking?

We are given a binary matrix:

0 → empty cell
1 → source cell

For every cell,

👉 find the shortest distance to the nearest cell containing 1.

Distance means:

Up
Down
Left
Right

movement only.

🌟 Example
Input
0 0 0
0 1 0
1 0 0
Output
1 1 2
1 0 1
0 1 2
🧠 Main Idea

Instead of starting BFS from every 0 ❌

We start BFS from ALL 1s together ✅

Because:

Nearest distance spreads outward level by level

This is:

Multi-Source BFS
⚡ Optimal Approach (BFS)
💡 Idea
Push all cells having 1 into queue.
Mark them visited.
Distance of those cells = 0.
Start BFS.
Neighbours get:
distance + 1
💻 Java Code
import java.util.*;

public class Main {

    static int[] row = {-1, 1, 0, 0};
    static int[] col = {0, 0, -1, 1};

    public static void main(String[] args) {

        int[][] grid = {
                {0,0,0},
                {0,1,0},
                {1,0,0}
        };

        int n = grid.length;
        int m = grid[0].length;

        int[][] distance = new int[n][m];

        boolean[][] visited = new boolean[n][m];

        Queue<int[]> queue = new LinkedList<>();

        // Push all 1s into queue
        for(int i=0;i<n;i++) {

            for(int j=0;j<m;j++) {

                if(grid[i][j] == 1) {

                    queue.offer(new int[]{i,j,0});

                    visited[i][j] = true;
                }
            }
        }

        while(!queue.isEmpty()) {

            int[] current = queue.poll();

            int r = current[0];
            int c = current[1];
            int dist = current[2];

            distance[r][c] = dist;

            for(int k=0;k<4;k++) {

                int newRow = r + row[k];
                int newCol = c + col[k];

                if(newRow>=0 && newRow<n &&
                   newCol>=0 && newCol<m &&
                   !visited[newRow][newCol]) {

                    visited[newRow][newCol] = true;

                    queue.offer(new int[]{
                            newRow,
                            newCol,
                            dist + 1
                    });
                }
            }
        }

        // Print answer
        for(int i=0;i<n;i++) {
            System.out.println(Arrays.toString(distance[i]));
        }
    }
}
▶️ Output
[1, 1, 2]
[1, 0, 1]
[0, 1, 2]
🌈 Dry Run
Initial Queue

All 1s inserted:

[(1,1,0), (2,0,0)]

Distance:

1-cells → 0
BFS Level 1

Neighbours get:

distance = 1

Cells:

(0,1)
(1,0)
(1,2)
(2,1)
(0,0)
BFS Level 2

Remaining cells:

(0,2)
(2,2)

Distance:

2

Done 🎉

⏱️ Time Complexity
O(N × M)

Because every cell visited once.

📦 Space Complexity
Auxiliary Space
O(N × M)

For:

queue
visited array
distance matrix
🔥 Important Points

✅ This is:

Multi-Source BFS

✅ All 1s start together.

✅ BFS guarantees shortest distance in unweighted grid.

✅ First time reaching a cell = shortest path.

🌟 Why BFS?

Because BFS explores:

Level by Level

So:

Level Number = Distance

Perfect for shortest distance problems 💖

🧠 Easy Memory Trick
Push all 1s
      ↓
Start BFS
      ↓
Spread outward
      ↓
Each level = distance
