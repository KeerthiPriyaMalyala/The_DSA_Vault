🚀 Shortest Path in Undirected Graph with Unit Weights
❓ What is the question asking?

We are given:

An Undirected Graph (Adjacency List)
Every edge has:
Weight = 1
A source node src

We need to find:

Shortest distance from source
to every other vertex.

If a node cannot be reached:

Distance = -1
🌟 Main Idea

Since every edge has the same weight:

Weight = 1

We can use:

BFS

Because BFS visits nodes:

Level by Level

Each level represents one extra edge.

So the first time we visit a node, we have already found its shortest distance.

⚡ Optimal Approach (BFS)
💡 Idea
Create adjacency list.
Create distance array.
Initialize all distances to:
∞ (Integer.MAX_VALUE)
Set:
distance[src] = 0
Push source into queue.
Perform BFS.
For every neighbor:
if(distance[node] + 1 < distance[neighbor])

update its distance and push it into the queue.

Convert all unreachable nodes (∞) to -1.
💻 Java Code
import java.util.*;

class Solution {

    public int[] shortestPath(int[][] edges, int n, int m, int src) {

        ArrayList<ArrayList<Integer>> adj = new ArrayList<>();

        for(int i = 0; i < n; i++) {
            adj.add(new ArrayList<>());
        }

        // Build adjacency list
        for(int i = 0; i < m; i++) {

            int u = edges[i][0];
            int v = edges[i][1];

            adj.get(u).add(v);
            adj.get(v).add(u);
        }

        int[] dist = new int[n];

        Arrays.fill(dist, Integer.MAX_VALUE);

        dist[src] = 0;

        Queue<Integer> q = new LinkedList<>();

        q.offer(src);

        while(!q.isEmpty()) {

            int node = q.poll();

            for(int neighbor : adj.get(node)) {

                if(dist[node] + 1 < dist[neighbor]) {

                    dist[neighbor] = dist[node] + 1;

                    q.offer(neighbor);
                }
            }
        }

        for(int i = 0; i < n; i++) {

            if(dist[i] == Integer.MAX_VALUE) {

                dist[i] = -1;
            }
        }

        return dist;
    }
}

⏱️ Time Complexity
O(V + E)

Where:

V = Number of vertices
E = Number of edges

Reason:

Every vertex is processed once.
Every edge is visited at most twice (undirected graph).
📦 Space Complexity
Auxiliary Space
O(V)

Used for:

Distance array
Queue

(Adjacency list stores the graph representation and is usually not counted as auxiliary space.)

🔥 Important Points

✅ Graph is Undirected.

✅ Every edge has weight = 1.

✅ BFS always finds the shortest path in an unweighted graph.

✅ First time a node is reached = shortest distance.

✅ If a node cannot be reached, return -1.
