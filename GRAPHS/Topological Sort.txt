🌳 Topological Sort (DFS)
❓ What is the question asking?

We are given a:

Directed Acyclic Graph (DAG)

We need to arrange nodes such that:

✅ If there is an edge:

u → v

then:

u must come BEFORE v

in ordering.

This ordering is called:

Topological Order
🌟 Example
Graph
5 → 0 ← 4
↓
2 → 3 → 1

One valid topo order:

5 4 2 3 1 0
🌟 Why?

Because:

5 comes before 0
5 comes before 2
2 comes before 3
3 comes before 1

All dependency rules satisfied ✅

🌟 Important Point

Topological Sort works ONLY for:

DAG

Meaning:

✅ Directed
✅ No cycles

🧠 Main Idea (DFS)

In DFS:

👉 A node should be added ONLY AFTER visiting all its neighbors.

So:

push node into stack AFTER DFS

This guarantees dependency order.

⚡ Optimal Approach (DFS + Stack)
💡 Idea
Start DFS.
Visit neighbors first.
Push node into stack after DFS finishes.
Pop stack to get topo order.
💻 Java Code
import java.util.*;

public class Main {

    static void dfs(int node,
                    ArrayList<ArrayList<Integer>> graph,
                    boolean[] visited,
                    Stack<Integer> stack) {

        visited[node] = true;

        for(int neighbor : graph.get(node)) {

            if(!visited[neighbor]) {

                dfs(neighbor,
                    graph,
                    visited,
                    stack);
            }
        }

        // Push after visiting neighbors
        stack.push(node);
    }

    public static void main(String[] args) {

        int V = 6;

        ArrayList<ArrayList<Integer>> graph =
                new ArrayList<>();

        for(int i=0;i<V;i++) {
            graph.add(new ArrayList<>());
        }

        graph.get(5).add(0);
        graph.get(5).add(2);

        graph.get(4).add(0);
        graph.get(4).add(1);

        graph.get(2).add(3);

        graph.get(3).add(1);

        boolean[] visited = new boolean[V];

        Stack<Integer> stack = new Stack<>();

        for(int i=0;i<V;i++) {

            if(!visited[i]) {

                dfs(i,graph,visited,stack);
            }
        }

        while(!stack.isEmpty()) {

            System.out.print(stack.pop() + " ");
        }
    }
}
▶️ Output
5 4 2 3 1 0

(Topo order may vary)

🔥 Important Points

✅ Node added AFTER visiting neighbors.

✅ Multiple valid topo orders possible.
