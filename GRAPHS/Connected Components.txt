💡 What is Connected Component?

aa group ni Connected Component antam 💖

Simple ga:

👉 “Mutually connected group of nodes”

💖 Important Point

👉 Connected Components concept mainly for:

🌸 Undirected Graphs

Because directed lo strongly/weakly connected concepts vastayi 😚

🌟 Logic

We use:

BFS OR
DFS

Idea:

Start from one unvisited node
Visit all reachable nodes
That forms ONE component 😎
Repeat for remaining unvisited nodes

💻 Java Code using DFS
import java.util.*;

public class Main {

    static void dfs(int node, ArrayList<ArrayList<Integer>> adj, boolean[] visited) {

        visited[node] = true;

        for(int neighbor : adj.get(node)) {

            if(!visited[neighbor]) {
                dfs(neighbor, adj, visited);
            }
        }
    }

    public static void main(String[] args) {

        int V = 5;

        ArrayList<ArrayList<Integer>> adj = new ArrayList<>();

        for(int i = 0; i < V; i++) {
            adj.add(new ArrayList<>());
        }

        // Component 1
        adj.get(0).add(1);
        adj.get(1).add(0);

        // Component 2
        adj.get(2).add(3);
        adj.get(3).add(2);

        // Node 4 isolated

        boolean[] visited = new boolean[V];

        int count = 0;

        for(int i = 0; i < V; i++) {

            if(!visited[i]) {

                dfs(i, adj, visited);

                count++;
            }
        }

        System.out.println("Connected Components = " + count);
    }
}
🌟 Output
Connected Components = 3
💙 Why count++ after DFS?

Because:

👉 One DFS visits ENTIRE connected group 😎✨

So:

one DFS = one component
🌸 Time Complexity

O(V + E)

Where:

V = vertices
E = edges
