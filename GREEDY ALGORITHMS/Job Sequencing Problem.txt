⏳ Job Sequencing Problem
❓ What is the question asking?

We are given jobs with:

id
deadline
profit

Each job takes:

1 unit of time

We need to:

✅ Do jobs in such a way that:

Job finishes before deadline
Total profit becomes maximum
🪨 Brute Force Approach
💡 Idea
Generate all possible job orders.
Check valid schedules.
Find maximum profit.

But permutations are huge 😵

Very slow for large inputs.

⚡ Optimal Approach (Greedy)
💡 Idea

✨ Greedy Thought:

Do the jobs with maximum profit first.

Steps:

Sort jobs by profit in descending order.
Try placing each job at the latest possible free slot before deadline.
If slot available:
do job
add profit

Why latest slot?

👉 So earlier slots remain available for other jobs.

💻 Java Code
import java.util.*;

class Job {
    int id;
    int deadline;
    int profit;

    Job(int id, int deadline, int profit) {
        this.id = id;
        this.deadline = deadline;
        this.profit = profit;
    }
}

public class Main {

    public static void jobSequencing(Job[] jobs) {

        // Sort by profit descending
        Arrays.sort(jobs, (a, b) -> b.profit - a.profit);

        // Find max deadline
        int maxDeadline = 0;

        for (Job job : jobs) {
            maxDeadline = Math.max(maxDeadline, job.deadline);
        }

        int[] slots = new int[maxDeadline + 1];

        Arrays.fill(slots, -1);

        int countJobs = 0;
        int totalProfit = 0;

        for (Job job : jobs) {

            // Check slots from deadline to 1
            for (int j = job.deadline; j > 0; j--) {

                if (slots[j] == -1) {

                    slots[j] = job.id;
                    countJobs++;
                    totalProfit += job.profit;
                    break;
                }
            }
        }

        System.out.println("Jobs Done = " + countJobs);
        System.out.println("Total Profit = " + totalProfit);
    }

    public static void main(String[] args) {

        Job[] jobs = {
                new Job(1, 4, 20),
                new Job(2, 1, 10),
                new Job(3, 1, 40),
                new Job(4, 1, 30)
        };

        jobSequencing(jobs);
    }
}
▶️ Output
Jobs Done = 2
Total Profit = 60
⏱️ Time Complexity
O(n log n + n × maxDeadline)
Sorting → O(n log n)
Slot checking → O(n × maxDeadline)
📦 Space Complexity
O(maxDeadline)

🧠 Important Greedy Point

✅ Always choose:

Highest profit job first

AND

✅ Place it at:

Latest possible slot

This helps maximize total profit 💖
