<13. Ninja and His Friends>
❓ Problem Understanding

A grid contains chocolates.

There are 2 friends (Alice & Bob).

Alice starts at (0,0).
Bob starts at (0,m-1) (top-right corner).

Every move, both go one row down.

Each friend can move:

↙ Down Left

↓

Down

↘ Down Right

Each collects chocolates from the cells they visit.

⚠️ If both stand on the same cell, count the chocolates only once.

Find the maximum chocolates they can collect.

Example
2   3   1   2

3   4   2   2

5   6   3   5

Alice

Start → (0,0)

Bob

Start → (0,3)

Need

Maximum Chocolates
🧠 Observation & Recurrence

In all previous Grid DP problems,

our state was

(i,j)

Now,

two people move simultaneously.

So one row is common, but both have different columns.

State becomes

(i, j1, j2)

Where

i  = current row

j1 = Alice's column

j2 = Bob's column

This is the biggest change in this problem.

Choices

Alice has 3 moves:

↙ ↓ ↘

Bob also has 3 moves:

↙ ↓ ↘

Total possibilities

3 × 3 = 9

At every state, try all 9 combinations.

Chocolate Collection

If

j1 == j2

Both are on the same cell.

Collect only once.

value = grid[i][j1];

Otherwise

value = grid[i][j1] + grid[i][j2];

🪨 Recursive Approach
💡 Idea

At every row,

both friends move.

Each has 3 options.

Try all 9 combinations.

Take the maximum.

💻 Java Code
public class Main {

    static int solve(int i, int j1, int j2, int[][] grid) {

        int n = grid.length;
        int m = grid[0].length;

        if (j1 < 0 || j1 >= m || j2 < 0 || j2 >= m)
            return -(int)1e9;

        if (i == n - 1) {

            if (j1 == j2)
                return grid[i][j1];

            return grid[i][j1] + grid[i][j2];
        }

        int maxi = -(int)1e9;

        for (int dj1 = -1; dj1 <= 1; dj1++) {

            for (int dj2 = -1; dj2 <= 1; dj2++) {

                int value;

                if (j1 == j2)
                    value = grid[i][j1];
                else
                    value = grid[i][j1] + grid[i][j2];

                value += solve(i + 1,
                        j1 + dj1,
                        j2 + dj2,
                        grid);

                maxi = Math.max(maxi, value);
            }
        }

        return maxi;
    }

    public static void main(String[] args) {

        int[][] grid = {

                {2,3,1,2},

                {3,4,2,2},

                {5,6,3,5}

        };

        System.out.println(solve(0,0,3,grid));
    }
}
