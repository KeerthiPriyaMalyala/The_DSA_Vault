66. Plus One
❓ What is the question asking?

You are given an integer represented as an array of digits.

Each element contains a single digit.
The number does not contain leading zeros.

Add 1 to the number and return the resulting array.

Example 1

Input:

digits = [1,2,3]

Output:

[1,2,4]

Explanation:

123 + 1 = 124
Example 2

Input:

digits = [4,3,2,1]

Output:

[4,3,2,2]
Example 3

Input:

digits = [9,9,9]

Output:

[1,0,0,0]
🪨 Brute Force Approach
💡 Idea
Convert the array into a number.
Add 1.
Convert the number back into an array.

❌ This approach is not practical because the number can be much larger than Java's long can store.

⚡️ Optimal Approach
💡 Idea

Start checking digits from the last index.

If the current digit is less than 9:
Increment it by 1.
Return the array.
Otherwise:
Make it 0 and continue to the previous digit (carry).
If all digits are 9, create a new array of size n + 1.
Put 1 at index 0.
Remaining elements are already 0.
💻 Java Code
class Main {

    public static int[] plusOne(int[] digits) {

        int n = digits.length;

        for (int i = n - 1; i >= 0; i--) {

            if (digits[i] < 9) {
                digits[i]++;
                return digits;
            }

            digits[i] = 0;
        }

        int[] ans = new int[n + 1];
        ans[0] = 1;

        return ans;
    }

    public static void main(String[] args) {

        int[] digits = {9, 9, 9};

        int[] result = plusOne(digits);

        for (int num : result)
            System.out.print(num + " ");
    }
}
▶️ Output
1 0 0 0
⏱️ Time Complexity
O(n)
In the worst case (all digits are 9), we traverse the entire array once.
📦 Space Complexity
O(1)
No extra space is used except when creating the answer array for all 9s.
📝 Dry Run

Example 2

[1,3,0]
Example 3

↓

9 → 0
↓

9 → 0

Create new array

📝 Interview Notes 💖
✅ Traverse the array from right to left.
✅ If a digit is less than 9, increment it and return immediately.
✅ If a digit is 9, change it to 0 and continue (carry).
✅ If all digits are 9, create a new array of size n + 1 with the first element as 1.
❌ Do not convert the array into an integer because the number can exceed Java's data type limits.
✅ Time Complexity: O(n)
✅ Space Complexity:
O(1) normally.
A new array of size n + 1 is created only when all digits are 9.
🎯 Interview One-Liner

Process the digits from right to left, handle the carry just like manual addition, and create a new array only if every digit is 9. 💖🚀
