724. Find Pivot Index
❓ What is the question asking?

Given an integer array nums, find the leftmost index such that:

Sum of all elements on the LEFT
=
Sum of all elements on the RIGHT

If no such index exists, return -1.

🪨 Brute Force Approach
💡 Idea

For every index:

Calculate the left sum.
Calculate the right sum.
If both are equal, return that index.
💻 Java Code
class Solution {
    public int pivotIndex(int[] nums) {

        for (int i = 0; i < nums.length; i++) {

            int leftSum = 0;
            int rightSum = 0;

            for (int j = 0; j < i; j++) {
                leftSum += nums[j];
            }

            for (int j = i + 1; j < nums.length; j++) {
                rightSum += nums[j];
            }

            if (leftSum == rightSum)
                return i;
        }

        return -1;
    }
}
▶️ Dry Run
nums = [1,7,3,6,5,6]

Left = 0

Right = 7+3+6+5+6 = 27

Not Equal

Index 1

Left = 1

Right = 3+6+5+6 = 20

Not Equal

Index 2

Left = 8

Right = 17

Not Equal

Index 3

Equal ✅

3
⏱️ Time Complexity
O(n²)
📦 Space Complexity
O(1)
⚡ Optimal Approach (Prefix Sum)
💡 Idea

Instead of calculating the left and right sums every time:

Find the total sum of the array.
Keep a running leftSum.
Right sum can be calculated as:
rightSum = totalSum - leftSum - nums[i]

If

leftSum == rightSum

then i is the pivot index.

💻 Java Code
class Solution {
    public int pivotIndex(int[] nums) {

        int totalSum = 0;

        for (int num : nums)
            totalSum += num;

        int leftSum = 0;

        for (int i = 0; i < nums.length; i++) {

            int rightSum = totalSum - leftSum - nums[i];

            if (leftSum == rightSum)
                return i;

            leftSum += nums[i];
        }

        return -1;
    }
}

At index i:

Total Sum

=

Left Sum

+

nums[i]

+

Right Sum

So,

Right Sum

=

Total Sum

-

Left Sum

-

nums[i]

=

Perfect! 🎉

🎯 Interview Notes
✅ Brute Force checks every index by recalculating left and right sums.
✅ Optimal first computes the total sum.
✅ Maintain leftSum.
✅ Calculate rightSum using:
rightSum = totalSum - leftSum - nums[i]
✅ Return the first (leftmost) pivot index.
✅ If no pivot exists, return -1.
🧠 Memory Trick
Total Sum

↓

Keep Left Sum

↓

Right Sum = Total - Left - Current

↓

If Left == Right

↓

Return Index

💖 Formula to remember forever:

Right Sum = Total Sum - Left Sum - Current Element

This one formula is the heart of the optimal solution. 🌸✨
