204. Count Primes
❓ What is the question asking?

Given an integer n, return the number of prime numbers strictly less than n.

A prime number has exactly 2 factors:

1
Itself
🪨 Brute Force Approach
💡 Idea
Check every number from 2 to n-1.
For each number, check whether it is prime by trying all divisors from 2 to number-1.
If it is prime, increase the count.
💻 Java Code
class Solution {

    public boolean isPrime(int num) {

        if (num <= 1)
            return false;

        for (int i = 2; i < num; i++) {

            if (num % i == 0)
                return false;
        }

        return true;
    }

    public int countPrimes(int n) {

        int count = 0;

        for (int i = 2; i < n; i++) {

            if (isPrime(i))
                count++;
        }

        return count;
    }
}
▶️ Output
Input: n = 10
Output: 4
⏱️ Time Complexity
O(n²)
📦 Space Complexity
O(1)
🟡 Better Approach
💡 Idea

Instead of checking divisors till num-1, check only till √num.

Because if a number has a factor greater than √num, it must also have a factor smaller than √num.

💻 Java Code
class Solution {

    public boolean isPrime(int num) {

        if (num <= 1)
            return false;

        for (int i = 2; i * i <= num; i++) {

            if (num % i == 0)
                return false;
        }

        return true;
    }

    public int countPrimes(int n) {

        int count = 0;

        for (int i = 2; i < n; i++) {

            if (isPrime(i))
                count++;
        }

        return count;
    }
}
▶️ Output
Input: n = 10
Output: 4
⏱️ Time Complexity
O(n√n)
📦 Space Complexity
O(1)
⚡️ Optimal Approach (Sieve of Eratosthenes)
💡 Idea
Assume every number is prime.
Start from 2.
If a number is prime, mark all its multiples as not prime.
Start marking from i × i.
Count all remaining prime numbers.
💻 Java Code
class Solution {

    public int countPrimes(int n) {

        if (n <= 2)
            return 0;

        boolean[] prime = new boolean[n];

        for (int i = 2; i < n; i++) {
            prime[i] = true;
        }

        for (int i = 2; i * i < n; i++) {

            if (prime[i]) {

                for (int j = i * i; j < n; j += i) {
                    prime[j] = false;
                }
            }
        }

        int count = 0;

        for (int i = 2; i < n; i++) {

            if (prime[i])
                count++;
        }

        return count;
    }
}
▶️ Output
Input: n = 10
Output: 4
⏱️ Time Complexity
O(n log log n)
📦 Space Complexity
O(n)
🧠 Why start from i * i?

Example: i = 5

5 × 2 = 10 ❌ Already marked by 2

5 × 3 = 15 ❌ Already marked by 3

5 × 4 = 20 ❌ Already marked by 2

5 × 5 = 25 ✅ Start here

So,

j = i * i

avoids unnecessary work.
