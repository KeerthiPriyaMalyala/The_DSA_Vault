Correct BST with Two Nodes Swapped 🌳💖
❓ What is the question asking?

Given:

A Binary Search Tree
Two nodes are swapped by mistake 😭

We must:

Restore the BST

without changing tree structure.

🌳 Example
Wrong BST ❌
        6
       / \
      10  2
     / \  / \
    1  3 7  12

Here:

10 and 2

are swapped.

🌟 Correct BST ✅
        6
       / \
      2   10
     / \  / \
    1  3 7  12
💡 Main BST Logic

Important BST property:

Inorder Traversal of BST is SORTED
🌟 Inorder of Wrong BST
1 10 3 6 7 2 12

This is NOT sorted ❌

We detect violations.

💡 Violation Rule

If:

prev.data > current.data

Then BST property is broken 🚨

🌟 Cases
Case 1 → Adjacent Swap

Example:

1 3 2 4

Only one violation.

Case 2 → Non-Adjacent Swap

Example:

1 10 3 6 7 2 12

Two violations 💖

🌟 Main Idea

During inorder traversal maintain:

prev
first
middle
last
First Violation

If:

prev > current

Store:

first = prev
middle = current
Second Violation

Again if violation occurs:

Store:

last = current
🌟 Final Fix
If NON-Adjacent swap

Swap:

first ↔ last
If Adjacent swap

Swap:

first ↔ middle
🌟 Dry Run

Wrong inorder:

1 10 3 6 7 2 12
Violation 1
10 > 3

Store:

first = 10
middle = 3
Violation 2
7 > 2

Store:

last = 2
Final Swap

Swap:

10 ↔ 2

BST restored ✅💖

⚡️ Optimal Approach
💻 Java Code
class Node {

    int data;
    Node left, right;

    Node(int data) {
        this.data = data;
        left = right = null;
    }
}

public class Main {

    static Node first = null;
    static Node middle = null;
    static Node last = null;
    static Node prev = null;

    // INORDER TRAVERSAL
    public static void inorder(Node root) {

        if (root == null) {
            return;
        }

        inorder(root.left);

        // VIOLATION DETECTED
        if (prev != null && prev.data > root.data) {

            // FIRST VIOLATION
            if (first == null) {

                first = prev;
                middle = root;
            }

            // SECOND VIOLATION
            else {

                last = root;
            }
        }

        prev = root;

        inorder(root.right);
    }

    // FIX BST
    public static void correctBST(Node root) {

        inorder(root);

        // NON-ADJACENT SWAP
        if (first != null && last != null) {

            int temp = first.data;
            first.data = last.data;
            last.data = temp;
        }

        // ADJACENT SWAP
        else if (first != null && middle != null) {

            int temp = first.data;
            first.data = middle.data;
            middle.data = temp;
        }
    }

    // PRINT INORDER
    public static void printInorder(Node root) {

        if (root == null) {
            return;
        }

        printInorder(root.left);

        System.out.print(root.data + " ");

        printInorder(root.right);
    }

    public static void main(String[] args) {

        // WRONG BST
        Node root = new Node(6);

        root.left = new Node(10);
        root.right = new Node(2);

        root.left.left = new Node(1);
        root.left.right = new Node(3);

        root.right.left = new Node(7);
        root.right.right = new Node(12);

        System.out.println("Before Fix:");

        printInorder(root);

        correctBST(root);

        System.out.println();

        System.out.println("After Fix:");

        printInorder(root);
    }
}
▶️ Output
Before Fix:
1 10 3 6 7 2 12

After Fix:
1 2 3 6 7 10 12
⏱️ Time Complexity
O(N)

Single inorder traversal 💖

📦 Space Complexity
O(H)

Recursive stack height 🌳
