Check if a Tree is BST or Not рҹҢівң…
вқ“ What is the question asking?

We need to check whether a binary tree follows BST rules or not.

BST Rule рҹ’–
Left subtree values < root.data
Right subtree values > root.data

AND

Every subtree should ALSO be BST
рҹҢҹ IMPORTANT TRICK

Just checking:

root.left.data < root.data
root.right.data > root.data

is NOT enough рҹҳӯвқҢ

Because violation can happen deeper.

Example:

        10
       /  \
      5    15
          /
         6

This is NOT BST вқҢ

Because:

6 is inside RIGHT subtree of 10
So 6 should be > 10

But:

6 < 10

Violation рҹҳӯ

вҡЎпёҸ Optimal Approach
рҹ’Ў Idea

For every node:

node value must lie between:
min range
max range

Initially:

(-вҲһ , +вҲһ)

When going LEFT:

max becomes root.data

When going RIGHT:

min becomes root.data
рҹҢі Visualization

For root = 10

Allowed range:
(-вҲһ , +вҲһ)

Left child:

(-вҲһ , 10)

Right child:

(10 , +вҲһ)
рҹ’» Java Code
class Node
{
    int data;
    Node left;
    Node right;

    Node(int data)
    {
        this.data = data;
        left = null;
        right = null;
    }
}

public class Main
{
    static boolean isBST(Node root, long min, long max)
    {
        // Empty tree is BST
        if(root == null)
        {
            return true;
        }

        // Violation
        if(root.data <= min || root.data >= max)
        {
            return false;
        }

        // Check left and right
        return isBST(root.left, min, root.data) &&
               isBST(root.right, root.data, max);
    }

    public static void main(String[] args)
    {
        Node root = new Node(10);

        root.left = new Node(5);
        root.right = new Node(15);

        root.right.left = new Node(6);
        root.right.right = new Node(20);

        boolean ans = isBST(root, Long.MIN_VALUE, Long.MAX_VALUE);

        System.out.println(ans);
    }
}

Because:
