🌟 Count Good Numbers
❓ What is the question asking?

We need to count how many strings of length n are possible such that:

✅ Even Index Positions

(0, 2, 4 ...)

Can contain only:

0, 2, 4, 6, 8

Total choices:

5 choices
✅ Odd Index Positions

(1, 3, 5 ...)

Can contain only prime digits:

2, 3, 5, 7

Total choices:

4 choices

We need:

✅ Total number of good strings.

Answer can become huge 😭

So return:

ans % 1000000007
🌈 Example
Input
n = 1

Only index 0 exists.

Even index → 5 choices.

Answer:

5
Input
n = 2

Index 0 → 5 choices
Index 1 → 4 choices

Total:

5 × 4 = 20
🧠 MOST IMPORTANT OBSERVATION

We DO NOT generate all strings 😭

We use maths directly 💖

🌟 Count Even & Odd Positions

Suppose:

n = 5

Indexes:

0 1 2 3 4

Even positions:

0,2,4

Count:

3

Odd positions:

1,3

Count:

2
🌟 Formula
Even positions
(n + 1) / 2
Odd positions
n / 2
🌟 Total Good Numbers
5^(evenCount) × 4^(oddCount)
⚡ Main Challenge

Power becomes HUGE 😭

So we use:

Binary Exponentiation

same as Pow(x,n) problem 💖

🌈 Dry Run
Input
n = 4
Even positions
(4+1)/2 = 2
Odd positions
4/2 = 2
Total
5² × 4²
=
25 × 16
=
400

Answer ✅

⚡ Optimal Approach
💻 Java Code
public class Main {

    static long MOD = 1000000007;

    // Binary Exponentiation
    public static long power(long x, long n) {

        // Base case
        if (n == 0) {
            return 1;
        }

        long half = power(x, n / 2);

        long result = (half * half) % MOD;

        // Odd power
        if (n % 2 == 1) {
            result = (result * x) % MOD;
        }

        return result;
    }

    public static int countGoodNumbers(long n) {

        long evenCount = (n + 1) / 2;
        long oddCount = n / 2;

        long evenWays = power(5, evenCount);
        long oddWays = power(4, oddCount);

        return (int)((evenWays * oddWays) % MOD);
    }

    public static void main(String[] args) {

        long n = 4;

        System.out.println(countGoodNumbers(n));
    }
}
▶️ Output
400
