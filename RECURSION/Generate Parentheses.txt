🌟 Generate Parentheses
❓ What is the question asking?

We need to generate all:

✅ Valid parenthesis combinations

using:

n pairs

of brackets.

🌈 Example
Input
n = 3
Output
((()))
(()())
(())()
()(())
()()()
❌ Invalid Strings
()))((
(()(()

Because parentheses are not balanced.

🧠 Main Recursion Idea

At every step we try:

(
or
)

BUT with rules 💖

🌟 Important Rules
✅ We can add "("

ONLY if:

open < n

because maximum n opening brackets allowed.

✅ We can add ")"

ONLY if:

close < open

Why?

Because:

closing brackets can never exceed opening brackets

Otherwise string becomes invalid 😭

🌈 MOST IMPORTANT THINKING

We NEVER generate invalid strings.

We stop invalid paths EARLY ✨

This is called:

Pruning

VERY important backtracking concept 💖

"("

⚡ Optimal Recursive Approach
💻 Java Code
public class Main {

    public static void generate(int n,
                                int open,
                                int close,
                                String current) {

        // Base case
        if (current.length() == 2 * n) {
            System.out.println(current);
            return;
        }

        // Add opening bracket
        if (open < n) {
            generate(n,
                     open + 1,
                     close,
                     current + "(");
        }

        // Add closing bracket
        if (close < open) {
            generate(n,
                     open,
                     close + 1,
                     current + ")");
        }
    }

    public static void main(String[] args) {

        int n = 3;

        generate(n, 0, 0, "");
    }
}
