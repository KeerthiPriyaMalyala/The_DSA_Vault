❓ What is the question asking?

atoi means:

ASCII to Integer

We are given a numeric string like:

"1234"

We need to convert it into:

✅ Recursion only

🧠 Recursive Thinking

⚡ Optimal Recursive Approach
💻 Java Code
public class Main {

    public static int atoi(String str, int n) {

        // Base case
        if (n == 0) {
            return 0;
        }

        // Recursive call for smaller string
        int smallAns = atoi(str, n - 1);

        // Current digit
        int digit = str.charAt(n - 1) - '0';

        // Build answer
        return smallAns * 10 + digit;
    }

    public static void main(String[] args) {

        String str = "1234";

        int ans = atoi(str, str.length());

        System.out.println(ans);
    }
}
▶️ Output
1234
⏱️ Time Complexity
O(n)

One recursive call for each character.

📦 Space Complexity
O(n)

Recursion stack space.
