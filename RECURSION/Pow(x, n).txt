⚡ Pow(x, n)
❓ What is the question asking?

We need to calculate:

xⁿ

Example:

2³ = 8

Using:

✅ Recursion

🪨 Normal Recursive Approach
💡 Idea

We know:

xⁿ = x × xⁿ⁻¹

Example:

2³
= 2 × 2²
= 2 × 2 × 2¹
= 2 × 2 × 2 × 2⁰
💻 Java Code
public class Main {

    public static long power(long x, long n) {

        // Base case
        if (n == 0) {
            return 1;
        }

        return x * power(x, n - 1);
    }

    public static void main(String[] args) {

        long x = 2;
        long n = 5;

        System.out.println(power(x, n));
    }
}
▶️ Output
32
⏱️ Time Complexity
O(n)
📦 Space Complexity
O(n)
⚡ Optimal Approach (Binary Exponentiation)
💡 Main Genius Idea

Instead of reducing:

n → n-1

we reduce:

n → n/2

This makes recursion SUPER FAST 😭💖

🌟 Important Maths
If n is EVEN
xⁿ = (x^(n/2))²

Example:

2⁴
= (2²)²
= 4²
= 16
If n is ODD
xⁿ = x × (x^(n/2))²

Example:

2⁵
= 2 × (2²)²
= 2 × 4²
= 32

💻 Optimal Java Code
public class Main {

    public static long power(long x, long n) {

        // Base case
        if (n == 0) {
            return 1;
        }

        long half = power(x, n / 2);

        // Even power
        if (n % 2 == 0) {
            return half * half;
        }

        // Odd power
        return x * half * half;
    }

    public static void main(String[] args) {

        long x = 2;
        long n = 5;

        System.out.println(power(x, n));
    }
}
▶️ Output
32
⏱️ Time Complexity
O(log n)

Because:

n becomes n/2 every time
📦 Space Complexity
O(log n)
