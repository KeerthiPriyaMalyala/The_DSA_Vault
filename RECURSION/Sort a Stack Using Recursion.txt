🧠 Main Recursion Idea

Exactly like reverse stack 💖

We use 2 recursive functions:

1️⃣ sortStack()

✅ Removes elements one by one

2️⃣ sortedInsert()

This function:

✅ Inserts element in correct sorted position.

⚡ Optimal Recursive Approach
💻 Java Code
import java.util.*;

public class Main {

    // Insert element in sorted order
    public static void sortedInsert(Stack<Integer> st, int data) {

        // Base case
        if (st.isEmpty() || st.peek() <= data) {
            st.push(data);
            return;
        }

        int top = st.pop();

        sortedInsert(st, data);

        st.push(top);
    }

    // Sort stack using recursion
    public static void sortStack(Stack<Integer> st) {

        // Base case
        if (st.isEmpty()) {
            return;
        }

        int top = st.pop();

        sortStack(st);

        sortedInsert(st, top);
    }

    public static void main(String[] args) {

        Stack<Integer> st = new Stack<>();

        st.push(3);
        st.push(1);
        st.push(4);
        st.push(2);

        System.out.println("Original Stack:");
        System.out.println(st);

        sortStack(st);

        System.out.println("Sorted Stack:");
        System.out.println(st);
    }
}

⏱️ Time Complexity
O(n²)

Why?

For every element:

may traverse stack again.

🌈 Difference Between Reverse Stack & Sort Stack
Problem	Insert Function
Reverse Stack	Insert at bottom
Sort Stack	Insert in sorted position
🧠 Golden Recursion Thinking

Whenever recursion problem comes:

Ask:

✅ What is smaller problem?

Here:

Sort remaining smaller stack

Then:

✅ What current work should I do?
Insert current element in correct position
