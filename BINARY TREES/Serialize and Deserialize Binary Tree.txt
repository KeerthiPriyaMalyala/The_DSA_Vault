Serialize and Deserialize Binary Tree 🌳💖
❓ What is the question asking?

We need to:

Convert tree ➜ String (Serialize)
Convert String ➜ Same tree (Deserialize)

We use:

Level Order Traversal (BFS)
# for NULL nodes
, as delimiter
🌳 Example Tree
        1
       / \
      2   3
         / \
        4   5
🔐 Serialization
💡 Idea
Use Queue
Traverse level by level
If node exists → store value
If node is null → store "#"
🪄 Serialized String
1,2,3,#,#,4,5,#,#,#,#,
🔓 Deserialization
💡 Idea
Split string using ,
Create root using first value
Use Queue
Attach left and right children one by one
🌟 Dry Run

Data:

1,2,3,#,#,4,5,#,#,#,#,
Step 1

Root = 1

Queue:

[1]
Step 2

Pop 1

Next values:

2 , 3

Tree:

    1
   / \
  2   3

Queue:

[2,3]
Step 3

Pop 2

Next values:

# , #

So 2 has no children.

Step 4

Pop 3

Next values:

4 , 5

Final Tree:

        1
       / \
      2   3
         / \
        4   5
⚡️ Optimal Approach
💻 Java Code
import java.util.*;

class Node {
    int data;
    Node left, right;

    Node(int data) {
        this.data = data;
        left = right = null;
    }
}

public class Main {

    // SERIALIZE
    public static String serialize(Node root) {

        if (root == null)
            return "";

        StringBuilder sb = new StringBuilder();

        Queue<Node> q = new LinkedList<>();
        q.offer(root);

        while (!q.isEmpty()) {

            Node temp = q.poll();

            // NULL NODE
            if (temp == null) {
                sb.append("#,");
                continue;
            }

            sb.append(temp.data).append(",");

            q.offer(temp.left);
            q.offer(temp.right);
        }

        return sb.toString();
    }

    // DESERIALIZE
    public static Node deserialize(String data) {

        if (data.equals(""))
            return null;

        String values[] = data.split(",");

        Node root = new Node(Integer.parseInt(values[0]));

        Queue<Node> q = new LinkedList<>();
        q.offer(root);

        int i = 1;

        while (!q.isEmpty()) {

            Node parent = q.poll();

            // LEFT CHILD
            if (!values[i].equals("#")) {

                Node leftNode =
                    new Node(Integer.parseInt(values[i]));

                parent.left = leftNode;
                q.offer(leftNode);
            }

            i++;

            // RIGHT CHILD
            if (!values[i].equals("#")) {

                Node rightNode =
                    new Node(Integer.parseInt(values[i]));

                parent.right = rightNode;
                q.offer(rightNode);
            }

            i++;
        }

        return root;
    }

    // LEVEL ORDER PRINT
    public static void levelOrder(Node root) {

        Queue<Node> q = new LinkedList<>();
        q.offer(root);

        while (!q.isEmpty()) {

            Node temp = q.poll();

            System.out.print(temp.data + " ");

            if (temp.left != null)
                q.offer(temp.left);

            if (temp.right != null)
                q.offer(temp.right);
        }
    }

    public static void main(String[] args) {

        Node root = new Node(1);

        root.left = new Node(2);
        root.right = new Node(3);

        root.right.left = new Node(4);
        root.right.right = new Node(5);

        // SERIALIZE
        String ans = serialize(root);

        System.out.println("Serialized Tree:");
        System.out.println(ans);

        // DESERIALIZE
        Node newRoot = deserialize(ans);

        System.out.println("\nLevel Order After Deserialize:");

        levelOrder(newRoot);
    }
}

Level Order After Deserialize:
1 2 3 4 5
