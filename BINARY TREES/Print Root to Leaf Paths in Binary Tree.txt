Print Root to Leaf Paths in Binary Tree

❓ What is the question asking?

You are given a binary tree.
You need to print all possible paths from:

Root → Leaf Node

A leaf node is a node having no left and right child.

Example:

        1
      /   \
     2     3
    / \     
   4   5

Root to leaf paths are:

1 -> 2 -> 4
1 -> 2 -> 5
1 -> 3
🪨 Brute Force Approach

💡 Idea

Use recursion
Maintain a list storing current path
Whenever a leaf node is reached:
Print the current path

This is the straightforward recursive solution.

💻 Java Code

import java.util.*;

class Node {
    int data;
    Node left, right;

    Node(int data) {
        this.data = data;
        left = right = null;
    }
}

public class Main {

    static void printPaths(Node root, ArrayList<Integer> path) {

        if (root == null)
            return;

        // Add current node
        path.add(root.data);

        // If leaf node, print path
        if (root.left == null && root.right == null) {

            for (int val : path) {
                System.out.print(val + " ");
            }

            System.out.println();
        }

        // Recur for left and right subtree
        printPaths(root.left, path);
        printPaths(root.right, path);

        // Backtracking
        path.remove(path.size() - 1);
    }

    public static void main(String[] args) {

        Node root = new Node(1);

        root.left = new Node(2);
        root.right = new Node(3);

        root.left.left = new Node(4);
        root.left.right = new Node(5);

        ArrayList<Integer> path = new ArrayList<>();

        printPaths(root, path);
    }
}

▶️ Output

1 2 4
1 2 5
1 3

⏱️ Time Complexity

O(N)

Every node is visited once.

📦 Space Complexity

O(H)

Where H is height of tree (recursion stack + path list).

⚡️ Optimal Approach

💡 Idea

The above recursive backtracking solution itself is the optimal approach because:

Every node must be visited at least once
Uses only one path list
Efficient backtracking avoids extra copies
Steps
Add current node to path
If leaf node → print path
Recur left
Recur right
Remove current node (backtrack)

💻 Java Code

import java.util.*;

class Node {
    int data;
    Node left, right;

    Node(int data) {
        this.data = data;
        left = right = null;
    }
}

public class Main {

    static void rootToLeaf(Node root, List<Integer> path) {

        if (root == null)
            return;

        path.add(root.data);

        // Leaf node
        if (root.left == null && root.right == null) {

            System.out.println(path);
        }

        rootToLeaf(root.left, path);
        rootToLeaf(root.right, path);

        // Backtrack
        path.remove(path.size() - 1);
    }

    public static void main(String[] args) {

        Node root = new Node(1);

        root.left = new Node(2);
        root.right = new Node(3);

        root.left.left = new Node(4);
        root.left.right = new Node(5);

        List<Integer> path = new ArrayList<>();

        rootToLeaf(root, path);
    }
}

▶️ Output

[1, 2, 4]
[1, 2, 5]
[1, 3]

⏱️ Time Complexity

O(N)

📦 Space Complexity

O(H)
