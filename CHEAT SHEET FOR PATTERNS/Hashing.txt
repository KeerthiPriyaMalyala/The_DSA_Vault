🧠 What is Hashing?

Hashing is simply storing information so that we can find it instantly.

Instead of searching every element (O(n)),

we store it in a HashMap/HashSet and get it in almost O(1) time.

Think of it like:

👧 "Where is 5?"

Without HashMap:

1 3 7 5 9
        ↑
Search one by one 😭

With HashMap:

5 → Found immediately 😎

🌸 Common Mistakes

❌ Forgetting

getOrDefault()

Instead writing

map.get()+1

which causes NullPointerException.

❌ Adding before checking duplicate

Wrong

set.add(num);

if(set.contains(num))

Always

Check

↓

Add

❌ Updating HashMap at the wrong time in Two Sum.

Always check first, then insert the current value.
