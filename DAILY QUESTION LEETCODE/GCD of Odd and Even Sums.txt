3658. GCD of Odd and Even Sums
❓ What is the question asking?

You are given an integer n.

Find:

sumOdd = Sum of the first n odd numbers.
sumEven = Sum of the first n even numbers.

Return:

gcd(sumOdd, sumEven)
Example 1

Input:

n = 4

Output:

4

Explanation:

Odd Numbers

1 + 3 + 5 + 7 = 16

Even Numbers

2 + 4 + 6 + 8 = 20

gcd(16,20)=4
Example 2

Input:

n = 5

Output:

5

Explanation:

Odd Numbers

1+3+5+7+9=25

Even Numbers

2+4+6+8+10=30

gcd(25,30)=5
🪨 Brute Force Approach
💡 Idea
Calculate the sum of the first n odd numbers.
Calculate the sum of the first n even numbers.
Use Euclid's Algorithm to find the GCD.
💻 Java Code
class Main {

    static int gcd(int a, int b) {

        while (b != 0) {

            int temp = b;
            b = a % b;
            a = temp;
        }

        return a;
    }

    static int gcdOfOddEvenSums(int n) {

        int sumOdd = 0;
        int sumEven = 0;

        for (int i = 1; i <= 2 * n; i++) {

            if (i % 2 == 0)
                sumEven += i;
            else
                sumOdd += i;
        }

        return gcd(sumOdd, sumEven);
    }

    public static void main(String[] args) {

        System.out.println(gcdOfOddEvenSums(4));
    }
}
▶️ Output
4
⏱️ Time Complexity
O(n)
📦 Space Complexity
O(1)
⚡️ Optimal Approach (Math)
💡 Idea

Instead of calculating the sums using a loop, use mathematical formulas.

Formula 1

The sum of the first n odd numbers is:

n²

Example:

1+3+5+7=16=4²
Formula 2

The sum of the first n even numbers is:

n(n+1)

Example:

2+4+6+8=20

4×5=20

Now we need:

gcd(n² , n(n+1))

Factor out n:

gcd(n×n , n×(n+1))

= n × gcd(n , n+1)

Since consecutive numbers are always coprime:

gcd(n,n+1)=1

Therefore,

Answer = n
