1979. Find Greatest Common Divisor of Array
❓ What is the question asking?

You are given an integer array nums.

Find:

The smallest element in the array.
The largest element in the array.
Return the GCD (Greatest Common Divisor) of these two numbers.

Note: You only need the GCD of the minimum and maximum elements, not the GCD of all array elements.

Example 1

Input:

nums = [2,5,6,9,10]

Output:

2

Explanation:

Smallest = 2

Largest = 10

gcd(2,10) = 2
Example 2

Input:

nums = [7,5,6,8,3]

Output:

1

Explanation:

Smallest = 3

Largest = 8

gcd(3,8) = 1
Example 3

Input:

nums = [3,3]

Output:

3
🪨 Brute Force Approach
💡 Idea
Find the minimum and maximum elements.
Check every number from 1 to min(min, max).
The largest number dividing both is the GCD.
💻 Java Code
class Main {

    static int gcd(int a, int b) {

        int ans = 1;

        for (int i = 1; i <= Math.min(a, b); i++) {

            if (a % i == 0 && b % i == 0)
                ans = i;
        }

        return ans;
    }

    static int findGCD(int[] nums) {

        int min = nums[0];
        int max = nums[0];

        for (int num : nums) {

            min = Math.min(min, num);
            max = Math.max(max, num);
        }

        return gcd(min, max);
    }

    public static void main(String[] args) {

        int nums[] = {2,5,6,9,10};

        System.out.println(findGCD(nums));
    }
}
▶️ Output
2
⏱️ Time Complexity
O(n + min(min,max))
📦 Space Complexity
O(1)
⚡️ Optimal Approach
💡 Idea
Find the minimum and maximum elements in one traversal.
Use Euclid's Algorithm to compute the GCD efficiently.

Euclid's Formula:

gcd(a,b) = gcd(b, a % b)

Repeat until b == 0.

💻 Java Code
class Main {

    static int gcd(int a, int b) {

        while (b != 0) {

            int temp = b;
            b = a % b;
            a = temp;
        }

        return a;
    }

    static int findGCD(int[] nums) {

        int min = nums[0];
        int max = nums[0];

        for (int num : nums) {

            min = Math.min(min, num);
            max = Math.max(max, num);
        }

        return gcd(min, max);
    }

    public static void main(String[] args) {

        int nums[] = {2,5,6,9,10};

        System.out.println(findGCD(nums));
    }
}
▶️ Output
2
⏱️ Time Complexity
O(n + log(max(nums)))
📦 Space Complexity
O(1)

📝 Interview Notes 💖
✅ Find the smallest and largest elements in the array.
✅ Compute the GCD of only these two numbers.
✅ Brute Force: Check every divisor up to min(min, max).
✅ Optimal: Use Euclid's Algorithm.
✅ Euclid's Formula:
gcd(a,b) = gcd(b, a % b)
✅ Time Complexity:
Brute Force → O(n + min(min,max))
Optimal → O(n + log(max(nums)))
✅ Space Complexity → O(1)
🎯 Interview One-Liner
