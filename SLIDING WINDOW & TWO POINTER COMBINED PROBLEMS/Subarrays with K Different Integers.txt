992. Subarrays with K Different Integers

is basically the same pattern as:

✅ 930. Binary Subarrays With Sum
✅ 1248. Count Number of Nice Subarrays

❓ What is the question asking?

Given an array:

nums = [1,2,1,2,3]
k = 2

Find the number of subarrays having exactly k distinct integers.

Example
[1,2]
[2,1]
[1,2]
[2,1,2]
[1,2,1]
[1,2,1,2]
[2,3]

Answer:

7
🧠 Main Observation

We cannot directly count:

Exactly K Distinct

using sliding window.

So we use:

Exactly K Distinct
=
At Most K Distinct
-
At Most (K-1) Distinct

Same formula as:

Exactly K Sum
=
At Most K Sum
-
At Most (K-1) Sum

and

Exactly K Odds
=
At Most K Odds
-
At Most (K-1) Odds
🪨 Brute Force Approach
💡 Idea

Generate all subarrays.

For every subarray:

count distinct integers using HashSet
if distinct count == k
→ increment answer
⏱️ Time Complexity
O(n²)
📦 Space Complexity
O(n)
⚡ Optimal Approach
💡 Idea

Create helper:

countAtMost(k)

which returns:

Number of subarrays
having at most k distinct integers

Then:

answer =
countAtMost(k)
-
countAtMost(k-1)
💻 Java Code
import java.util.HashMap;

class Solution {

    private int countAtMost(int[] nums, int k) {

        int left = 0;
        int count = 0;

        HashMap<Integer,Integer> map = new HashMap<>();

        for(int right = 0; right < nums.length; right++) {

            map.put(
                nums[right],
                map.getOrDefault(nums[right], 0) + 1
            );

            while(map.size() > k) {

                map.put(
                    nums[left],
                    map.get(nums[left]) - 1
                );

                if(map.get(nums[left]) == 0) {
                    map.remove(nums[left]);
                }

                left++;
            }

            count += right - left + 1;
        }

        return count;
    }

    public int subarraysWithKDistinct(int[] nums, int k) {

        return countAtMost(nums, k)
             - countAtMost(nums, k - 1);
    }
}
▶️ Output
nums = [1,2,1,2,3]
k = 2

Output = 7
⏱️ Time Complexity
O(n)
📦 Space Complexity
O(k)
