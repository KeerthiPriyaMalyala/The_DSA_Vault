1248. Count Number of Nice Subarrays
❓ What is the question asking?

You are given:

an integer array nums
an integer k

A subarray is called nice if it contains exactly k odd numbers.

Find the number of nice subarrays.

Example

Input:

nums = [1,1,2,1,1]
k = 3

Output:

2

Explanation:

Nice subarrays are:

[1,1,2,1]
[1,2,1,1]

Answer = 2

🪨 Brute Force Approach
💡 Idea

Generate all possible subarrays.

For every subarray:

count odd numbers
if odd count equals k
→ increment answer
💻 Java Code
public class Main {

    public static int numberOfSubarrays(int[] nums, int k) {

        int count = 0;

        for(int i = 0; i < nums.length; i++) {

            int oddCount = 0;

            for(int j = i; j < nums.length; j++) {

                if(nums[j] % 2 != 0) {
                    oddCount++;
                }

                if(oddCount == k) {
                    count++;
                }
            }
        }

        return count;
    }

    public static void main(String[] args) {

        int[] nums = {1,1,2,1,1};

        int k = 3;

        System.out.println(numberOfSubarrays(nums, k));
    }
}
▶️ Output
2
⏱️ Time Complexity
O(n²)
📦 Space Complexity
O(1)
⚡ Optimal Approach
💡 Idea

This is exactly the same pattern as:

930. Binary Subarrays With Sum

Replace:

1 → odd number
0 → even number

Use:

Count(Exactly K Odds)
=
Count(At Most K Odds)
-
Count(At Most (K - 1) Odds)

Create a helper function:

countAtMost(k)

which counts subarrays having at most k odd numbers.

💻 Java Code
public class Main {

    public static int countAtMost(int[] nums, int k) {

        if(k < 0) {
            return 0;
        }

        int left = 0;
        int oddCount = 0;
        int count = 0;

        for(int right = 0; right < nums.length; right++) {

            if(nums[right] % 2 != 0) {
                oddCount++;
            }

            while(oddCount > k) {

                if(nums[left] % 2 != 0) {
                    oddCount--;
                }

                left++;
            }

            count += right - left + 1;
        }

        return count;
    }

    public static int numberOfSubarrays(int[] nums, int k) {

        return countAtMost(nums, k)
                - countAtMost(nums, k - 1);
    }

    public static void main(String[] args) {

        int[] nums = {1,1,2,1,1};

        int k = 3;

        System.out.println(numberOfSubarrays(nums, k));
    }
}
▶️ Output
2
⏱️ Time Complexity
O(n)
📦 Space Complexity
O(1)
🧠 Pattern Connection
Binary Subarrays With Sum (930)
1 = contributes to sum
0 = does not contribute
Count Number of Nice Subarrays (1248)
Odd = contributes to odd count
Even = does not contribute

Both use the exact same formula:

Count(Exactly K)
=
Count(At Most K)
-
Count(At Most (K - 1))
