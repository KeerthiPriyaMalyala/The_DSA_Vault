3. Longest Substring Without Repeating Characters

❓ What is the question asking?

Given a string s.

We need to find the length of the longest substring
that contains no repeating characters 💖

Substring means:

continuous characters only

Example:

s = "abcabcbb"

Longest substring without repeating characters:

"abc"

Length = 3

Answer = 3

🪨 Brute Force Approach
💡 Idea

Generate all substrings.

For every substring:

check if all characters are unique
if yes, update maximum length
💻 Java Code
import java.util.HashSet;

public class Main {

    public static int lengthOfLongestSubstring(String s) {

        int n = s.length();
        int maxLen = 0;

        for (int i = 0; i < n; i++) {

            HashSet<Character> set = new HashSet<>();

            for (int j = i; j < n; j++) {

                char ch = s.charAt(j);

                if (set.contains(ch)) {
                    break;
                }

                set.add(ch);

                maxLen = Math.max(maxLen, j - i + 1);
            }
        }

        return maxLen;
    }

    public static void main(String[] args) {

        String s = "abcabcbb";

        System.out.println(lengthOfLongestSubstring(s));
    }
}
▶️ Output
3
⏱️ Time Complexity
O(n²)
📦 Space Complexity
O(256) ≈ O(1)
⚡ Optimal Approach (Sliding Window + HashSet)
💡 Idea

Use two pointers:

left
right

Maintain a window with unique characters.

If duplicate comes:

remove characters from left
until duplicate is removed

Keep updating maximum length 💖

💻 Java Code
import java.util.HashSet;

public class Main {

    public static int lengthOfLongestSubstring(String s) {

        HashSet<Character> set = new HashSet<>();

        int left = 0;
        int maxLen = 0;

        for (int right = 0; right < s.length(); right++) {

            char ch = s.charAt(right);

            // Remove until duplicate gone
            while (set.contains(ch)) {

                set.remove(s.charAt(left));

                left++;
            }

            set.add(ch);

            maxLen = Math.max(maxLen, right - left + 1);
        }

        return maxLen;
    }

    public static void main(String[] args) {

        String s = "abcabcbb";

        System.out.println(lengthOfLongestSubstring(s));
    }
}
▶️ Output
3
⏱️ Time Complexity
O(n)
📦 Space Complexity
O(256) ≈ O(1)
